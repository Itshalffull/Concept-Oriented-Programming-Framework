# GroupSpec

Define named grouping specifications that cluster records into hierarchical groups and compute aggregations over each group . Each group spec captures a serialized GroupingConfig ( type , fields , and optional set memberships ) , an AggregationDef [ ] describing functions and aliases to compute per group , and an optional post aggregation FilterNode for HAVING style pruning . Group specs can be evaluated directly against a JSON row array , returning a structured result containing the group key , the matching rows , and the computed aggregates for each bucket

## Commands

### create
Register a new named group spec with its serialized GroupingConfig and 
 AggregationDef array . The spec is immediately available for evaluation 
 and retrieval .

**Arguments:** `$0` **name** (string), `$1` **grouping** (string), `$2` **aggregations** (string)

### get
Return the group spec identifier , its serialized GroupingConfig , 
 AggregationDef array , and optional HAVING filter for the given name .

**Arguments:** `$0` **name** (string)

### evaluate
Apply the named group spec s GroupingConfig to the JSON serialized row 
 array , compute each AggregationDef per bucket , and apply the optional 
 HAVING filter to prune groups . The result is a JSON array where each 
 entry contains the group key , the matching rows , and the computed 
 aggregates : [ { key , rows , aggregates } ] .

**Arguments:** `$0` **name** (string), `$1` **rows** (string)

### list
Return a JSON serialized array of all registered group spec names .
